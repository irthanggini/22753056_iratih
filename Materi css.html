CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan dan tata letak halaman web. Ini adalah salah satu teknologi inti dalam pengembangan web modern. Dengan menggunakan CSS, Anda dapat mengontrol warna, ukuran, jenis font, jarak, dan berbagai aspek tampilan lainnya dari elemen-elemen pada halaman web. CSS memisahkan tampilan dan struktur konten, sehingga memungkinkan Anda untuk dengan mudah mengubah penampilan halaman web secara konsisten pada seluruh situs.

Berikut adalah beberapa konsep utama dalam CSS:

Selektor:

CSS menggunakan selektor untuk menentukan elemen mana yang akan diberikan gaya. Misalnya, Anda dapat menggunakan selektor elemen (`p` untuk paragraf) atau selektor kelas (`.kelas` untuk elemen dengan kelas tertentu).

Properti:

 Properti CSS mengontrol aspek-aspek tertentu dari elemen yang dipilih. Contoh properti termasuk `color` (warna teks), `font-size` (ukuran font), dan `margin` (jarak dari elemen sekitarnya).

Value:

Value digunakan untuk mengatur properti. Misalnya, nilai `red` dapat digunakan untuk mengatur warna teks menjadi merah, dan nilai `20px` dapat digunakan untuk mengatur ukuran margin menjadi 20 piksel.

Aturan CSS:

Aturan CSS terdiri dari selektor, diikuti oleh blok properti dan nilai yang terkait

Kelas dan ID:

 Kelas dan ID adalah atribut yang dapat ditambahkan ke elemen HTML untuk memilih dan menerapkan gaya tertentu melalui CSS. Kelas digunakan dengan selektor kelas (`.kelas`), sementara ID digunakan dengan selektor ID (`#id`).

Perwarisan dan keutamaan:

 CSS menggunakan model perwarisan, di mana gaya diteruskan dari elemen induk ke elemen anak. Namun, jika ada konflik, aturan keutamaan digunakan untuk menentukan gaya mana yang akan diterapkan.

Posisi dan Tata Letak:

 CSS juga memiliki kemampuan untuk mengatur posisi dan tata letak elemen pada halaman web. Anda dapat menggunakan properti seperti `position` (posisi elemen), `display` (cara elemen ditampilkan), dan `float` (mengatur aliran elemen).
