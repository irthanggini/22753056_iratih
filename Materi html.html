HTML (HyperText Markup Language)

HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membangun dan mengatur struktur konten di halaman web. Ini adalah komponen dasar dari setiap situs web dan digunakan untuk menggambarkan elemen-elemen seperti teks, gambar, video, tautan, tabel, dan formulir. HTML menggunakan tag-tag (elemen) untuk mengelompokkan dan mengidentifikasi bagian-bagian tertentu dari halaman web. Setiap tag dimulai dengan tanda "<" dan diakhiri dengan tanda ">". Beberapa tag memiliki elemen yang terkait di dalamnya, yang dibungkus dengan tag pembuka dan penutup.

Struktur Dasar:

Sebuah dokumen HTML dimulai dengan elemen `html` yang mengelilingi seluruh konten. Di dalamnya, terdapat elemen `head` yang berisi informasi tentang halaman seperti judul dan stylesheet. Elemen `body` berisi konten aktual yang akan ditampilkan di halaman.

Elemen:

 HTML terdiri dari berbagai elemen yang digunakan untuk memformat dan mengorganisir konten. Elemen-elemen ini diapit oleh tag pembuka `tag` dan tag penutup `/tag`. Contoh elemen umum termasuk `h1` hingga `h6` untuk judul, `p` untuk paragraf, `a` untuk tautan, dan `img` untuk gambar.

Atribut:

 Elemen-elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. Misalnya, atribut `href` pada elemen `a` digunakan untuk menentukan URL tautan, dan atribut `src` pada elemen `img` menentukan sumber gambar.

Daftar:

 HTML menyediakan elemen untuk membuat daftar, baik daftar terurut (ordered) maupun tidak terurut (unordered). Elemen `ul` digunakan untuk daftar tidak terurut dengan menggunakan elemen `li` sebagai item dalam daftar. Elemen `ol` digunakan untuk daftar terurut.

Pengaturan Tabel:

 Elemen `table`, `tr`, dan `td` digunakan untuk membuat tabel. `table` mendefinisikan tabel, `tr` mendefinisikan baris, dan `td` mendefinisikan sel dalam tabel.

Beberapa elemen umum yang sering digunakan dalam HTML antara lain:
- `h1` hingga `h6`: digunakan untuk judul dengan tingkatan yang berbeda.
- `p`: digunakan untuk paragraf teks.
- `img`: digunakan untuk menampilkan gambar.
- `a`: digunakan untuk membuat tautan (link) ke halaman web lain.
- `ul` dan `li`: digunakan untuk membuat daftar dengan tanda bulet.
- `table`, `tr`, `td`: digunakan untuk membuat tabel dan baris-kolom data.
- `form` dan `input`: digunakan untuk membuat formulir dan menerima input pengguna.
